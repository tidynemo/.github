# nemo-verse

R packages for tidying bioinformatic pipeline outputs.

## 🐢 [nemo](https://github.com/umccr/nemo)

Building blocks for parsing, tidying, and writing bioinformatic pipeline results.  
📚 [Docs](https://umccr.github.io/nemo) · 📦 [Conda](https://anaconda.org/umccr/r-nemo)

## 🧬 [tidywigits](https://github.com/umccr/tidywigits)

Parse and tidy outputs from the [WiGiTS (hmftools)](https://github.com/hartwigmedical/hmftools) suite for cancer genomics.  
📚 [Docs](https://umccr.github.io/tidywigits) · 📦 [Conda](https://anaconda.org/umccr/r-tidywigits)
