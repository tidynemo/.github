## Overview

This is a GitHub organization profile repository for the **TidyNemo** (`tidynemo`) organization. It contains only the public-facing profile page displayed on the organization's GitHub page.

## Repository Structure

- `profile/README.md` — the organization profile displayed at `github.com/tidynemo`. This is the only content file that matters.
- `CODE_OF_CONDUCT.md` — Contributor Covenant v2.1.

## Purpose

The profile README showcases R packages for tidying bioinformatic pipeline outputs. Changes here are purely editorial — updating package descriptions, badges, links, or the table of repositories. There are no build steps, tests, or CI pipelines in this repo itself.
